Eslint 是一个 js 代码检查工具，可以用来避免低级的语法错误以及统一代码风格。

## 安装

### 自动安装并配置

```
npm init @eslint/config
```

执行上述命令就能进入到 Eslint 的配置程序，我们可以根据提示选择

![](附件/image/Eslint的使用_image_2.png)





```
npm init @eslint/config
```
